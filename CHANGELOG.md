# Changelog

All notable changes to FairPay will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changed
- Documentation migrated to Serena MCP knowledge base
- Removed 170+ markdown documentation files from repository
- Knowledge now stored in `.serena/memories/` for AI-assisted development

### Fixed
- Performance optimizations with Lighthouse improvements
- OAuth security fixes
- Individual split settlement functionality
- Receipt storage and display

## [2.0.0] - 2025-12-30

### Added
- Complete expense sharing functionality
- Group and friend expense tracking
- Individual split settlement with partial payment support
- Receipt attachments
- Real-time balance updates
- Google OAuth authentication
- Multi-language support (English, Vietnamese)
- Donation widget
- Public demo mode
- Historical transactions view

### Changed
- Migrated to shadcn/ui components
- Improved mobile responsiveness
- Enhanced UX with hr.refine.dev design patterns

### Technical
- React 19.1.0 + TypeScript 5.8.3
- Refine v5.0.1 framework
- Supabase backend
- Tailwind CSS v4.1.4
- Custom SVG icon system
