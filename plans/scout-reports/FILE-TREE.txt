DATA FLOW
=========

Queries:
  ├─ useTable (groups) → GroupList
  ├─ useOne (group/:id) → GroupShow, GroupEdit
  ├─ useList (group_members) → MemberList
  ├─ useList (expenses) → Balance calculation
  ├─ useList (payments) → Balance calculation
  ├─ useList (friendships) → GroupCreate, AddMemberModal
  └─ useSimplifiedDebts → GroupShow (debts simplification)

Mutations:
  ├─ useCreate (groups) → GroupCreate
  ├─ useCreate (group_members) → AddMemberModal
  ├─ useUpdate (groups) → GroupEdit
  ├─ useDelete (groups) → GroupShow
  └─ useDelete (group_members) → MemberList
